# SmartPick

## How to set up
1. Download and install Ollama from
   - https://ollama.com" and
   - run "ollama pull mistral" in terminal
2. At backend run
   - pip install -r requirements.txt
   - python app.py
3. Then at the frontend run
   - npm install
   - npm run build
4. Open "http://localhost:5000" to view.

Note: Even if Ollama is not installed, the system automatically falls back to rule-based responses but it is not recommended.
